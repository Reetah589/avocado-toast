Primitive Data Types:

Non-Primitive Data Types

Variable:

var: used throughtout the whole program, can be changed let: will only be used within the scope of where you declare const: is a variable that can't change

If/Else/Else If

if: Use the statement to specify a block of JavaScript code to be executed if a condition is true. else: Use the statement to specify a block of code to be executed if the condition is false. else if: Use this statement to specify a new condition if the first condition is false.

For Loop/ While Loop

while loop: this loops through a block of code as long as a specified condition is true for loop: this loops thorugh a block of code a number of times

Switch

The switch statement is uesed to perform different actions based on different conditions. The value of the expression is compared with the values of each case. There is a default code added and if there is no match, the default block code is executed.

Try / Catch

try: this statement lets you test a block of code for errors. catch: this statement lets you handle the error in the try block of code. throw: this statement lets you create custom errors.

Promises

The promise in JavaScript is an object that represents the eventual completion (or failure) of an asynchronus operation and its resulting value.
